# Usage
```bash
// React
cd frontend
npm install http-proxy-middleware --save  
npm install axios --save  
npm start  

// Flask
cd Backend/src/main/python
python -m virtualenv env
env/Scripts/activate.bat
pip install -r requests.txt
python run.py
```

Spring Server : 8080  
React Server : 3000  
Flask Server : 5000
